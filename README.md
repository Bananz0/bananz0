# Hey, I'm Glen 

Electrical & Electronics Engineering student at the University of Southampton, specializing in embedded systems, digital design, and the intersection of hardware and software.


## Tech Stack

**Hardware & Digital Design**
- IC Design: Cadence S-Edit/L-Edit/T-Spice, Mentor Calibre (DRC/LVS), GDSII
- FPGA: SystemVerilog, Altera Quartus, Xilinx Vivado, ModelSim
- PCB: KiCad, EAGLE, LTSpice
- Embedded: ESP32, STM32, Arduino, Raspberry Pi

**Software Development**
- Languages: C/C++, Python, JavaScript/TypeScript, Kotlin, C#/.NET
- Frameworks: React.js, Node.js, Qt, Android SDK
- Protocols: MQTT, Modbus, I2C/SPI/UART, AirPlay

**Infrastructure & Security**
- Networking: OPNsense, UniFi, Tailscale, HAProxy
- DevOps: Docker, Proxmox, Prometheus/Grafana
- Security: IDS/IPS (Suricata, Zenarmor), VPN configs, VLAN segmentation

## Current Interests

- **Space Systems** - Flight software development for CubeSats following NASA/MISRA C standards
- **Digital Design** - Full-custom IC design, FPGA architectures, and hardware acceleration
- **IoT Security** - SmartThings/HomeKit/Matter protocol implementation and mesh network security
- **Amateur Radio** - Studying for Foundation License, exploring SDR and RF design
